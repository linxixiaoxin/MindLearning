# 全书导读

> 领导者真正的瓶颈，往往不在知识和技能不够，而在心智结构不足以承载复杂世界。

## 这本书在回答什么

- 它讨论的不是“如何让领导者多学几招”，而是“如何让领导者长出更复杂的意义建构能力”。
- 它没有停在[[成人发展理论]]本身，而是一路推进到教练、培训、反馈、会议、组织设计，最后把“工作本身”改写成成长场。
- 如何让领导发展从知识技能升级，进入心智结构的成长，从而承载复杂、动态、多利益相关者的现实。

## 全书怎么展开

- 第一部分《渐趋复杂的成长之路》：建立[[成人发展理论]]总框架，并让结构判断从抽象理论变成可识别工具。
- 第二部分《协助他人成长》：把结构理论推进到个体教练、团体发展与持续练习。
- 第三部分《在工作中转化》：把成长直接嵌入领导工作与组织日常。

## 先抓住的关键推进

1. 复杂世界对领导者提出的，不只是更多知识，而是更大的心智容量。
2. 不同人之所以面对同一问题会反应完全不同，是因为他们在用不同心智结构建构现实。
3. 成人发展最值得关注的，不是内容正确与否，而是意义建构结构如何运作。
4. [[规范主导]]、[[自主导向]]、[[内观自变]]，代表了承载复杂性的不同层级。
5. 如果看不见结构，只盯着内容，就会把很多发展性问题误判成态度问题、能力问题或沟通问题。
