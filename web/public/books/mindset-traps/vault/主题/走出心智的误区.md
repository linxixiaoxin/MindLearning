## 这张主题页回答什么

1. 为什么曾经帮助人类生存的本能直觉，如今反而成了解决复杂问题的陷阱？
2. 人在面对复杂和不确定性时最容易掉入哪些隐蔽的思维误区？
3. 为什么[[妥协]]、达成共识和试图掌控，在复杂系统中往往会导向平庸甚至灾难？
4. 我们应该通过哪些反直觉的提问与习惯，重新训练大脑，在复杂局势中做出更好的决策？

## 核心逻辑

1. 人类大脑的操作系统是为简单、缓慢、线性因果的旧世界进化而来的。
2. 现代社会高度互联、变化迅速且充满不确定，旧操作系统难以处理复杂问题。
3. 旧本能在现代环境中异化为五大心智误区：[[简化故事]]、[[感觉正确]]、[[渴求共识]]、[[渴望掌控]]、捍卫自我。
4. 这些误区根植于生理与心理需求中，所以人们常常深陷其中却毫无自觉。
5. 要跳出误区，必须使用反直觉的行为工具，例如[[为了学习而倾听]]、在问题边缘做试验、放弃绝对控制。
6. 更深层的改变来自心智结构进化，即从[[规范主导]]、[[自主导向]]走向[[内观自变]]。
7. 当人能与人生目标、[[身体信号]]、[[情感颗粒度]]和[[仁爱之心]]建立联结时，才真正具备在复杂世界中行动的领导力。

## 建议一起看的 N 卡

- [[复杂性]]：复杂性：问题的难，不只是变量多，而是变量彼此影响、反馈滞后、因果非线性，无法被单点控制和一次看清。
- [[简化故事]]：简化故事：大脑为了更快理解世界，会自动补全缺失信息，把复杂现实压缩成一个看起来合理、但往往过于单一的叙事。
- [[感觉正确]]：感觉正确：一种强烈的确定感，让人把“我现在非常确信”误当成“事实已经被我看清”。
- [[为了学习而倾听]]：为了学习而倾听：不是为了反驳、说服或立刻解决问题而听，而是先进入对方如何理解现实的方式，让自己对问题的理解变得…
- [[社会疼痛]]：社会疼痛：人一旦感到被拒绝、被排斥、被孤立或被公开对立时，大脑会产生近似物理疼痛的威胁体验。
- [[渴求共识]]：渴求共识：在冲突、不确定和社会疼痛面前，人会本能地把“尽快一致”放到“把问题看完整”之前。
