# 所谓“反社会少年”，可能只是被错过了成长桥梁

> 问题不只在行为本身，而在环境要求的成熟度超前于个体当前的意义结构。

## 这张卡在回答什么

- 现实痛点：家长、老师和机构经常把难管的青少年直接当成坏、懒或不负责任。
- 章节范围：第六章唯我性自我的消长
- 为什么值得独立拿出来：它能直接回应家长和教育场景中最容易出现的误判。

- [[唯我性自我]]
- [[确认—矛盾—连续性]]
- [[植入的文化]]

## 关键关系

- 先被承认，才能被带向合作。
- 只做奖惩和控制，常常会帮旧平衡续命。
- 桥梁型环境比高压型环境更能把人送往下一阶段。

## 最适合从哪里展开

- 适合从 [[第06章 唯我性自我的消长]] 切进去。
- 一个少年在病房里被当成问题人物，在船厂里却慢慢学会合作。
- 你以为他只是自私，其实他还活在“只有我的需要最真实”的结构里。
- 行为矫正看起来有效，却像教鸽子打保龄球一样没有真正改变人。
