# 用「分后半：守住目标与领悟承诺」处理关键问题

> 好的收尾，会让客户带着责任感离开谈话。

## 什么时候适合用

- 期望结果必须被不断澄清。

## 这能解决什么问题

- 好的收尾，会让客户带着责任感离开谈话。

## 可以怎么做

1. 期望结果必须被不断澄清。
2. 领悟出现后要及时追问意义和下一步。
3. 没有承诺的领悟，只是一次舒服的感动。
4. 客户说“我知道了”却没说要做什么

## 使用边界

- 它不是一句万能话术，而是一套依赖关系安全、当下感和持续练习的对话方法。
