# 《第五项修炼学习型组织的艺术与实践》选题角度

## 一、可直接展开的小红书选题

### 1. 《第五项修炼学习型组织的艺术与实践》全书骨架：从起点问题到最终落点

- 选题类型：全书骨架型
- 对应章节：全书
- 现实痛点：组织里总在头痛医头脚痛医脚
- 核心矛盾：读者容易把「组织里总在头痛医头脚痛医脚」当成表面问题，但这本书真正要处理的是「系统思考」与「个人修炼」之间的结构关系。
- 必保概念：系统思考、个人修炼、心智模式、共同愿景、团队学习
- 必保关系：组织中的很多麻烦不是单点失误，而是结构持续制造的结果。；如果只盯事件层，就会不断救火而看不见系统回路。；系统思考帮助人看到延迟、反馈和相互依赖。；个人修炼、心智模式、共同愿景和团队学习则帮助组织建立新的学习能力。
- 候选场景：大家都在救火，却没人看结构。、学习的标准是回路有没有变。、很多决策失误源自只看局部。
- 推荐分页：首图总纲 + 章节地图 + 中轴主链
- 案例调用策略：按需调用案例，优先选最能说明主矛盾的 1 个
- 为什么值得做：最适合保住整本书的主问题、推进链和最终落点。
- 候选案例：
