# 用「分：如何内化教练思维」处理关键问题

> 教练先稳住自己，客户才可能稳住自己。

## 什么时候适合用

- 教练越想表现专业，越容易错过现场。

## 这能解决什么问题

- 教练先稳住自己，客户才可能稳住自己。

## 可以怎么做

1. 教练越想表现专业，越容易错过现场。
2. 教练必须先把自己的神经系统带回稳态。
3. 高阶教练的第一课不是多学技术，而是少让自我抢戏。
4. 去自我化

## 使用边界

- 它不是一句万能话术，而是一套依赖关系安全、当下感和持续练习的对话方法。
