# 傻瓜式选择
> 创建时间：2026-04-22
> 最后更新时间：2026-04-22

这是一个 `N / Note` 概念卡。

## 核心定义

`傻瓜式选择`：把关键对话误看成只能二选一的假命题，例如“说真话就会伤关系，不伤关系就只能闭嘴”。

## 可直接调用的判断

- 傻瓜式选择最典型的表现不是说错话，而是根本找不到第三条路。
- 它一旦出现，人很快就会滑向沉默或暴力，因为脑子里已经没有“既坦诚又尊重”的选项。
- 很多高风险场景真正需要打破的，不是对方，而是你心里那个假二选一框架。

## 这张卡解决什么

- 帮助你看见自己为何在关键时刻总想逃避或硬顶。
- 帮助你把“非此即彼”重写成“有没有同时保住两边的做法”。
- 帮助你从惯性反应进入更有创造力的对话。

## 进入哪些问题场景

- 向上沟通时觉得“说了会得罪领导，不说会害自己”。
- 亲密关系里觉得“坦白就要翻脸，不翻脸就只能憋着”。
- 团队里觉得“要么我赢，要么对方赢”。

## 场景动作模板

### 当你开始卡在二选一时

- 先把脑中那句“要么……要么……”写出来。
- 再问：如果我既想坦诚，又想保住关系，第三种做法可能是什么？
- 然后把这个问题交给真正目标和安全感工具，而不是直接交给情绪。

### 当你准备开口前

- 先澄清自己最怕失去什么。
- 再澄清自己真正想保住什么。
- 最后用对比法或事实开场去寻找第三条路。

## 红线边界

- 不要把现实约束很强的场景神化成“任何事都能完美两全”，重点是寻找更优而不是完美。
- 不要以为识别了傻瓜式选择，就等于已经找到新做法。
- 不要拿它当拖延工具，真正的价值在于打开行动空间。
