# Jennifer Garvey Berger

> **《走出心智的误区》作者**：领导力、组织行为学、认知心理学、成人发展理论

## 这本书最核心的贡献

- 人类为了应对简单环境而进化出的本能直觉，在复杂的现代世界中已经失效，并演变成了将我们引入歧途的五大心智误区。
- 本书不止解释认知偏差，还将成人发展理论引入领导力实战，强调从[[自主导向]]跃迁到[[内观自变]]。

## 它直接解决什么问题

- 在复杂和不确定环境中降低决策误判
- 改善团队冲突与沟通失效
- 减少控制欲带来的管理内耗
- 帮助个人突破心智停滞和自我防御
