# 用「从控制到教练」处理关键问题

> 适合转成图文、信息图和高频场景案例。

## 什么时候适合用

- 少一点指令，多一点提问

## 这能解决什么问题

- 适合转成图文、信息图和高频场景案例。

## 可以怎么做

1. 少一点指令，多一点提问
2. 不是放任，而是更高水平的管理
3. 很多绩效问题不是因为员工不听话，而是因为对话方式太控制。
4. 为什么你越爱管，团队反而越不长大

## 使用边界

- 它更像帮助校正判断与表达的框架，不适合被当成机械公式照搬。
