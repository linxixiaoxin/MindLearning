# 全书导读

> 这本书要解决的不是怎样提出更漂亮的问题，而是如何用“回放式探究”让客户看见自己的思维、情绪和模式，从而自己长出洞见与承诺。

## 这本书在回答什么

- 主问题：为什么发问不等于探究
- 核心转向：单纯发问很容易让教练忙着想问题，反而忽略眼前这个人。
- 最终落点：没有承诺收束的领悟很容易蒸发，因此好教练必须把领悟转成行动。

## 全书怎么推进

1. 教练不是专家给答案，而是思维伙伴帮助客户更独立地思考。
2. 单纯发问很容易让教练忙着想问题，反而忽略眼前这个人。
3. 回放式陈述能把客户说过的话、表现出的情绪和矛盾重新还给客户。
4. 当回放与提问结合时，客户更容易产生领悟而不是只给出答案。
5. 五大实践帮助教练聚焦人、守住目标、打开思维并推动行动。
6. 三大思维习惯让教练保持当下感、真正接收信息并释放评判。
