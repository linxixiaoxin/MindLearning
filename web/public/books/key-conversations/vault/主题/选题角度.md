# 《关键对话：如何高效能沟通（原书第2版）（珍藏版）》选题角度

## 一、可直接展开的小红书选题

### 1. 《关键对话：如何高效能沟通（原书第2版）（珍藏版）》全书骨架：从起点问题到最终落点

- 选题类型：全书骨架型
- 对应章节：全书
- 现实痛点：一到关键场合就失声或爆炸
- 核心矛盾：读者容易把「一到关键场合就失声或爆炸」当成表面问题，但这本书真正要处理的是「关键对话」与「共享意义池」之间的结构关系。
- 必保概念：关键对话、共享意义池、沉默与暴力、安全感、共同目的
- 必保关系：关键对话最难的不是内容，而是场上的风险、情绪和关系张力。；一旦人感到不安全，就会下意识走向沉默或暴力。；要让对话继续，先要恢复共同目的和互相尊重。；随后把事实、故事、感受和结论拆开，让信息重新回到共享池中。
- 候选场景：一到关键时刻就不是沉默就是爆炸。、没有安全感，再对的内容也进不去。、双方掌握的信息根本没被摆到台面。
- 推荐分页：首图总纲 + 章节地图 + 中轴主链
- 案例调用策略：按需调用案例，优先选最能说明主矛盾的 1 个
- 为什么值得做：最适合保住整本书的主问题、推进链和最终落点。
- 候选案例：
