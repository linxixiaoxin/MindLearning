# 第四章渴求共识的心智误区

> **这一章在全书里的功能**：拆解第三个误区，说明人类为什么害怕分歧。

## 这一章在解决什么

人类会为避免[[社会疼痛]]而过度追求一致。

## 核心论点

- 人类会为避免[[社会疼痛]]而过度追求一致。
- 盲目共识会压扁创造力和真实表达。
- 冲突并不总是风险，它也可能是理解升级的入口。

## 典型例子

- 团队表面和气，实则没有真问题被拿上台面

## 现实含义

- 把冲突重新定义为加深关系和拓展第三方案的机会。

## 关联概念

- [[社会疼痛]]

## 回到 K / N 主干

- [[走出心智的误区]]：人类为了应对简单环境而进化出的本能直觉，在复杂的现代世界中已经失效，并演变成了将我们引入歧途的五大心智误区。
- [[社会疼痛]]：社会疼痛：人一旦感到被拒绝、被排斥、被孤立或被公开对立时，大脑会产生近似物理疼痛的威胁体验。
