# 全书导读

> 这本书要解决的不是怎样把教练做得更流畅，而是如何在对话中创造真正的“顿悟时刻”：教练先调到深度临在，再帮助客户从旧叙事中产生新认知，并把洞见转成承诺。

## 这本书在回答什么

- 主问题：突破性时刻是如何发生的
- 核心转向：教练若想创造突破，先要把自己调到临在、开放、非反应的状态。
- 最终落点：洞见必须被说清、被承接并转成承诺，才可能变成长期改变。

## 全书怎么推进

1. 多数人知道很多道理却很难改变，因为大脑更想守住旧故事与安全区。
2. 教练若想创造突破，先要把自己调到临在、开放、非反应的状态。
3. 深度倾听和镜像反馈能让客户看见自己正在如何构造现实。
4. 情绪变化和沉默常常是顿悟逼近的信号，而不是会谈出问题。
5. 可观测目标会给会谈一个清晰支点，让探索不至于发散。
6. 客户真正的阻力往往藏在其操作系统、社交需求和价值观冲突里。
