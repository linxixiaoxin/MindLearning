# 情感颗粒度

## 核心定义

`情感颗粒度`：用更细、更准的语言区分自己到底在经历什么情绪，而不是一律叫作“难受”“烦”“压力大”。

## 核心观点

- 情绪命名越粗，行动选择越少；情绪命名越细，调整空间越大。
- 很多人不是不会处理情绪，而是根本没分清自己到底是在生气、羞耻、害怕、委屈、挫败，还是失望。
- 当情绪只剩一个模糊大包时，人更容易直接被情绪推着走。
- 所以这张卡不是语言洁癖，而是复杂判断能力的一部分。

## 这个概念在解决什么

- 帮助减少“被模糊情绪整体接管”的情况。
- 帮助把身体反应、情绪体验和后续行动重新拆开。
- 帮助在压力、冲突和自我防御里，保留更细的选择空间。

## 你会在什么情况下遇到它

- 你只会说“我很烦”，但其实里面混着焦虑、羞耻、委屈和不被理解。
- 一次冲突结束后，你知道自己状态不对，却说不清为什么会那么强烈。
- 团队里大家只会说“有压力”，但没人能讲清真正的问题在哪里。

## 你可以怎么使用

### 现场怎么用

- 当你意识到自己情绪上来了，先别急着判断对错，先把它拆细一点。
- 例如，不要只说“我不爽”，试着分成“我有点被轻视”“我有点慌”“我在担心失控”“我其实是羞耻”。
- 一旦命名更具体，后面的行动通常也会更准确。

### 复盘怎么用

- 事后复盘时，不只问“我怎么又发作了”，还要问“我当时具体在经历什么”。
- 这一步能帮你把旧模式从模糊阴影，变成可被识别的结构。

### 带人怎么用

- 如果别人只会说“我最近状态不好”，可以帮他继续拆：更像累、怕、怒、丢脸，还是没希望。
- 带人时不要急着给方案，先把情绪轮廓描清楚。

## 使用边界

- [[情感颗粒度]]不是让人沉迷分析情绪，也不是要求每次都说得极其准确。
- 它的重点不是把语言说漂亮，而是让判断和行动不再被一个模糊的大情绪整体吞掉。
- 如果一个人当前已经明显过载，先稳住，再细分，不要把“命名更细”变成额外压力。
