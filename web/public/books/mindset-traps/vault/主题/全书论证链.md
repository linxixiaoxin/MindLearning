# 全书论证链

> 用 7 步把《走出心智的误区》从根问题推进到最终落点。

## 主线

1. 人类大脑的操作系统是为简单、缓慢、线性因果的旧世界进化而来的。
2. 现代社会高度互联、变化迅速且充满不确定，旧操作系统难以处理复杂问题。
3. 旧本能在现代环境中异化为五大心智误区：[[简化故事]]、[[感觉正确]]、[[渴求共识]]、[[渴望掌控]]、捍卫自我。
4. 这些误区根植于生理与心理需求中，所以人们常常深陷其中却毫无自觉。
5. 要跳出误区，必须使用反直觉的行为工具，例如[[为了学习而倾听]]、在问题边缘做试验、放弃绝对控制。
6. 更深层的改变来自心智结构进化，即从[[规范主导]]、[[自主导向]]走向[[内观自变]]。
7. 当人能与人生目标、[[身体信号]]、[[情感颗粒度]]和[[仁爱之心]]建立联结时，才真正具备在复杂世界中行动的领导力。
