# 用「心智误区的由来」处理关键问题

> **类型**：判断原则 · **适用时机**：现代复杂世界与人类旧本能之间存在结构性不匹配。

## 怎么做

1. 现代复杂世界与人类旧本能之间存在结构性不匹配。
2. 很多失败不是努力不够，而是直觉系统已经过时。
3. 大脑是为简单、稳定的环境进化的，不是为复杂系统设计的。
4. 用过时操作系统处理新任务

## 希望带来什么效果

遇到困局时，先检查自己的思维系统是否适配环境。

## 使用边界

不适用于标准化、可预测、流程稳定的简单环境。
