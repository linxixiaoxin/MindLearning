# 第六章捍卫自我的心智误区

> **这一章在全书里的功能**：引入成人发展理论，解释最深层的防御来源。

## 这一章在解决什么

人会高估过去的改变，低估未来继续变化的可能。

## 核心论点

- 人会高估过去的改变，低估未来继续变化的可能。
- 很多人投入大量精力维护既有自我形象。
- 心智发展会经历[[规范主导]]、[[自主导向]]到[[内观自变]]的跃迁。

## 典型例子

- 在组织里打[[第二份工作]]，努力维持完美人设

## 现实含义

- 少问我是谁，多问我准备成为什么样的人。

## 关联概念

- [[第二份工作]]
- [[规范主导]]
- [[自主导向]]
- [[内观自变]]

## 回到 K / N 主干

- [[走出心智的误区]]：人类为了应对简单环境而进化出的本能直觉，在复杂的现代世界中已经失效，并演变成了将我们引入歧途的五大心智误区。
- [[第二份工作]]：第二份工作：人在组织里额外投入的大量隐形精力，用来掩饰弱点、维护形象、避免暴露和保护自我。
- [[规范主导]]：规范主导：一个人主要借助外部期待、群体规范和他人评价来组织自我与世界的心智结构。
- [[自主导向]]：自主导向：一个人开始用自己的原则、价值和内部系统来组织经验、做判断并维持行动方向的心智结构。
- [[内观自变]]：内观自变：一个人不仅能用自己的系统理解世界，也能把自己的系统当成观察对象，容纳多重视角并持续更新自我结构的心智…
