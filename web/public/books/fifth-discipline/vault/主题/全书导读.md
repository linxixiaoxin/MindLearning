# 全书导读

> 这本书的核心不是教企业开培训会，而是解释组织为什么会在自己的结构里不断制造问题，以及怎样通过系统思考和学习机制改变这种循环。

## 这本书在回答什么

- 主问题：什么是学习型组织？
- 核心转向：如果只盯事件层，就会不断救火而看不见系统回路。
- 最终落点：学习型组织的本质，是让集体不再重复制造同一种问题。

## 全书怎么推进

1. 组织中的很多麻烦不是单点失误，而是结构持续制造的结果。
2. 如果只盯事件层，就会不断救火而看不见系统回路。
3. 系统思考帮助人看到延迟、反馈和相互依赖。
4. 个人修炼、心智模式、共同愿景和团队学习则帮助组织建立新的学习能力。
5. 真正的成长不是某次培训，而是组织逐渐拥有自我觉察和自我修正能力。
6. 学习型组织的本质，是让集体不再重复制造同一种问题。
