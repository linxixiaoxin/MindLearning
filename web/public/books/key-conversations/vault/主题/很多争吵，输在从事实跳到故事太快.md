# 很多争吵，输在从事实跳到故事太快
> 创建时间：2026-04-22
> 最后更新时间：2026-04-22

这是一个 `K / Knowledge` 主题页。

**核心问题**：为什么很多冲突不是输在事实太糟，而是输在我们太快给事实编了故事，并被这个故事推着行动？

## 建议阅读路径

1. 从 [真正目标](../../2_N_concepts/N_notes/真正目标.md) 进入，先稳住对话方向。
2. 再看 [从事实到故事](../../2_N_concepts/N_notes/从事实到故事.md)，理解情绪和故事如何放大误解。
3. 最后接 [STATE表达法](../../2_N_concepts/N_notes/STATE表达法.md)，看怎样把危险判断安全地说出来。

## 这张主题页回答什么

1. 为什么同一件事，不同人会怒气值完全不同？
2. 受害者、大反派、无助者故事为什么会让对话越走越偏？
3. 怎样把“我就是这么感觉的”拆回到更完整的事实与责任？
4. 为什么先梳理故事，再表达观点，效果差别会这么大？

## 常见误判

- 以为情绪就是事实的直接反应。
- 以为只要我很愤怒，就说明我一定有理。
- 以为表达时直接抛结论更高效。
- 以为控制情绪就是把它压下去。

## 核心逻辑

1. 事实本身不会自动决定情绪，故事才会。
2. 一旦把故事当事实，就会把判断当证据，把情绪当真相。
3. 真正目标能帮你不被“想赢 / 想罚 / 想逃”的故事带走。
4. 从事实到故事的链条理清后，STATE 才不会变成包装过的攻击。
5. 所以很多争吵真正的转折点，不在更会说，而在更慢一点地下结论。

## 什么时候这条判断最有用

- 明知自己情绪太满，但又说不清为什么这么满。
- 你总感觉某些冲突一开口就像旧账自动接管。
- 你不是没有事实，而是很难分清哪些是事实、哪些是你脑中补完的解释。
