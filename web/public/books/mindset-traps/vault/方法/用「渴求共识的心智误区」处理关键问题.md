# 用「渴求共识的心智误区」处理关键问题

> **类型**：判断原则 · **适用时机**：人类会为避免[[社会疼痛]]而过度追求一致。

## 怎么做

1. 人类会为避免[[社会疼痛]]而过度追求一致。
2. 盲目共识会压扁创造力和真实表达。
3. 冲突并不总是风险，它也可能是理解升级的入口。
4. 团队表面和气，实则没有真问题被拿上台面

## 希望带来什么效果

把冲突重新定义为加深关系和拓展第三方案的机会。

## 使用边界

解释的是主观痛感机制，不代表存在真实外部危险。
