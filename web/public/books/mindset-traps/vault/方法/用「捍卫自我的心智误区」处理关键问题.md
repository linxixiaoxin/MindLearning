# 用「捍卫自我的心智误区」处理关键问题

> **类型**：判断原则 · **适用时机**：人会高估过去的改变，低估未来继续变化的可能。

## 怎么做

1. 人会高估过去的改变，低估未来继续变化的可能。
2. 很多人投入大量精力维护既有自我形象。
3. 心智发展会经历[[规范主导]]、[[自主导向]]到[[内观自变]]的跃迁。
4. 在组织里打[[第二份工作]]，努力维持完美人设
