# 关键习惯：表达不同意见来拓展可能性

> **类型**：行为习惯 · **适用时机**：高层争论反复僵住、各方都能拿出数据自证、但没有人真正知道最佳答案时。

## 怎么做

1. 先区分当前问题是可预测问题，还是没有现成标准答案的复杂问题。
2. 如果是复杂问题，不把讨论目标设成找到唯一正确答案，而是先扩大可能性空间。
3. 把不同意见表达成新线索、新风险或新试法，而不是表达成立场对撞。
4. 将彼此分歧转化成多个小范围、低成本、反馈快的试验方向。
5. 约定在试验后回看结果，用学习推进判断，而不是靠会场气势决定胜负。

## 希望带来什么效果

把不同意见从制造摩擦，转成拓展可能路径和并行试验的入口。

## 使用边界

不适用于必须立刻拍板的紧急场景，也不适合无限并行而没有回看机制。
