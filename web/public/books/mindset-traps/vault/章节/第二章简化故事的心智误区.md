# 第二章简化故事的心智误区

> **这一章在全书里的功能**：拆解第一个核心误区，解释人如何自动编造意义。

## 这一章在解决什么

大脑本能追求开头、结尾和清晰因果。

## 核心论点

- 大脑本能追求开头、结尾和清晰因果。
- 人会自动补全缺失信息并深信不疑。
- 我们会把过去投射到未来，用故事替代复杂现实。

## 典型例子

- 对同一事件可以编出多个完全不同但都自洽的故事

## 现实含义

- 在判断人和事时，强迫自己至少生成三个版本的解释。

- [[简化故事]]

## 回到 K / N 主干

- [[走出心智的误区]]：人类为了应对简单环境而进化出的本能直觉，在复杂的现代世界中已经失效，并演变成了将我们引入歧途的五大心智误区。
- [[简化故事]]：简化故事：大脑为了更快理解世界，会自动补全缺失信息，把复杂现实压缩成一个看起来合理、但往往过于单一的叙事。
