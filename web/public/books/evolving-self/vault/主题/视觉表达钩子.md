# 《发展的自我：自我成长中的过程与问题》视觉钩子

## 一、整本书可复用的视觉接口

### 1. 《发展的自我》成长螺旋总图

- 适用范围：全书
- 推荐结构：中心螺旋 + 六阶段节点 + 外圈支持环境
- 要表达的内容：把主体—客体关系的演化、每个自我阶段的危机，以及支持环境的变化画在一张总图里。
- 必须出现的对象：螺旋阶梯、主体与客体分界线、六个阶段节点、支持环境外环
- 最适合的页型：首图、总纲页、全书地图页
- 为什么适合做图：这本书最大的难点是结构过深，总图能一次性把读者带进框架。

### 2. 主体—客体关系对照图

- 适用范围：第一章至第三章
- 推荐结构：左右对照 + 阶段递进箭头
- 要表达的内容：一边画“我直接就是它”，另一边画“我开始能看见它”，让发展差异变成可视化对照。
- 必须出现的对象：镜子、眼睛、分界线、高瘦杯和矮宽杯、爆米花盒
- 最适合的页型：机制页、重构页、对照页
- 为什么适合做图：主体—客体分化很抽象，但一旦被画成左右两栏，理解速度会大幅提高。

### 3. 被支持着长大：抱持性环境时间轴

- 适用范围：第四章与第九章
- 推荐结构：时间轴 + 关键环境节点 + 三功能说明
- 要表达的内容：把抚育者、家庭、学校、同伴、工作和亲密关系画成一条连续的抱持性环境时间轴。
- 必须出现的对象：时间轴、母婴抱持、家庭、学校、工作、亲密关系、三角支撑
- 最适合的页型：时间线页、机制页、结尾收束页
- 为什么适合做图：自然疗法的力量就在于它不是一个点，而是一条贯穿一生的支持带。

### 4. 旧自我失效时会发生什么

- 适用范围：第六章至第八章
- 推荐结构：旧壳破裂 + 新自我未成形的中间带
- 要表达的内容：把反社会、恋家、空心感和价值混乱都画成“旧壳裂开但新结构尚未稳定”的中间地带。
- 必须出现的对象：裂开的外壳、桥梁、悬空的人影、关系线、规则网格
- 最适合的页型：危机页、转化页、桥梁页
- 为什么适合做图：它能帮助读者把“我是不是坏掉了”改读成“我是不是正在过渡”。

## 二、按章节拆页时可优先抓的对象

- 第一章未被认识的让·皮亚杰的才华：高瘦杯、矮宽杯、爆米花盒、帝国大厦楼顶的孩子
- 第二章道德意义建构的演化：棒球场、角色镜像、公平天平、两难选择卡
- 第三章自我的结构：螺旋发展图、主体—客体坐标轴、六阶段阶梯、外骨骼/内骨骼对照
- 第四章一体化自我的消长：母婴抱持、羊水环境、三角支撑结构、断裂又重新连接的依恋线
- 第五章冲动性自我的消失：角色面具、被封住的秘密盒、幻想怪物、儿童自制阶梯
- 第六章唯我性自我的消长：病房与船厂对照、交易天平、保龄球鸽子、需要之网
- 第七章人际性自我的消长：宿舍电话线、离家地图、关系与自我双轨图、被剪断又延续的脐带
- 第八章法规性自我的消长：裂开的规则网格、两极张力弧线、身份外壳、悖论环
- 第九章自然疗法：抱持性环境时间轴、家庭—学校—工作支持网络、确认—矛盾—连续性三角、心理羊水
