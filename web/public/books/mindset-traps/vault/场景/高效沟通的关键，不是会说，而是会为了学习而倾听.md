# 高效沟通的关键，不是会说，而是会为了学习而倾听

- 沟通越多误解越深，关系越来越僵。

## 建议先看哪几张 N 卡

- [[为了学习而倾听]]：为了学习而倾听：不是为了反驳、说服或立刻解决问题而听，而是先进入对方如何理解现实的方式，让自己对问题的理解变得…
- [[感觉正确]]：感觉正确：一种强烈的确定感，让人把“我现在非常确信”误当成“事实已经被我看清”。

## 为什么它容易被记住

兼具方法感和可执行性，适合做实用内容。

## 建议先点开的章节

- [[第一章心智误区的由来]]：现代复杂世界与人类旧本能之间存在结构性不匹配。
- [[第三章感觉正确的心智误区]]：确定感是一种情绪体验，而不是逻辑结论。
- [[第五章渴望掌控的心智误区]]：复杂系统无法被完全预测和控制。
