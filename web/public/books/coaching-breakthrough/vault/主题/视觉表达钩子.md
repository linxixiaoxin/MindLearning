# 《教练的突破》视觉钩子

## 一、整本书可复用的视觉接口

### 1. 《教练的突破》全书总图

- 适用范围：全书
- 推荐结构：中心总图 + 四周章节路径
- 要表达的内容：把「突破性时刻是如何发生的」画成一张能继续翻页的总图。
- 必须出现的对象：回环箭头、节点、流量管道、杠杆、仪表盘、透镜、拼图、路线图
- 最适合的页型：首图、总纲页、封面页
- 为什么适合做图：适合保住整本书的主问题、总结构和章节推进。

### 2. 临在 -> 倾听 -> 情绪变化 -> 洞见 -> 承诺。

- 适用范围：全书
- 推荐结构：突破性会谈路径图
- 要表达的内容：临在 -> 倾听 -> 情绪变化 -> 洞见 -> 承诺。
- 必须出现的对象：岔路口、轨道、锁链、时间轴、脚印、人物剪影、护盾、光圈
- 最适合的页型：首图、总纲页、目录页
- 为什么适合做图：适合把抽象概念快速变成一眼能懂的结构图。

### 3. 分：如何内化教练思维章节图

- 推荐结构：分：如何内化教练思维主结构 + 证据卡 + 底部收口
- 要表达的内容：教练越想表现专业，越容易错过现场。
- 必须出现的对象：回环箭头、节点、流量管道、杠杆、仪表盘、透镜
- 最适合的页型：章节页、机制页、信息图页
- 为什么适合做图：它能把这一章最值钱的关系线和现实后果一起画出来。
