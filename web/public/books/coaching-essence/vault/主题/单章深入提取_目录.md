# 《教练的本质》单章深入提取目录

- 这组资产按“每章一份”的方式重构，优先保住章节位置、核心问题、概念、方法与关系线。
- 它不是章节摘要的重复版，而是后续继续拆 `K` 卡、`N` 卡、多书站节点与内容页的中间层资产。
