# 旧脑子遇上新世界：为什么复杂环境会放大直觉误判

**核心问题**：为什么那些曾经帮人快速生存的直觉，在复杂环境里反而更容易把人带偏？

## 建议阅读路径

1. 先看 [[走出心智的误区]]，拿到全书主问题。
2. 再调用 [[复杂性]]、[[结构性错配]]、[[简化故事]]、[[感觉正确]]。
3. 最后补上 [[默认正确感]]、[[决定后的理由]]、[[反证过滤]]，把误判机制看完整。

## 主问题链

1. 为什么人不是更努力就能处理复杂问题，而是常常越努力越错？
2. 为什么复杂环境会让大脑更想讲清晰故事、更快找到确定感？
3. 为什么个体误判会继续升级成群体里的共识、掌控和身份防御？
4. 如果问题在旧操作系统失配，新操作系统又该往哪里升级？

## 用于调用的核心判断

- 很多失败不是能力不够，而是判断系统还在按旧世界的规则工作。
- 复杂环境放大的，不只是任务难度，也包括人类对简单因果、快速确定和稳定身份的依赖。
- 真正关键的升级，不是更快下判断，而是更早发现“环境已经变了，我的直觉却还没变”。

## 这张主题页回答什么

1. 为什么复杂世界会放大人的五大心智误区？
2. 这些误区为什么不是零散偏差，而是一套连续长出来的链条？
3. 从“直觉失灵”到“心智升级”，最值得先抓的入口是什么？

## 常见误判

- 以为复杂只是信息更多、工作更忙，本质上还是同一种问题。
- 以为只要更理性、更努力，就能压住直觉系统带来的偏差。
- 以为五大心智误区彼此独立，实际上它们经常是一环接一环地升级。

## 先看什么，再看什么

1. [[复杂性]]：先分清这不是“变量变多”，而是因果结构已经变了。
2. [[结构性错配]]：再看问题不只出在动作，而是环境与操作系统不匹配。
3. [[简化故事]]：接着看大脑如何把复杂现实压成单线叙事。
4. [[感觉正确]]、[[默认正确感]]、[[决定后的理由]]、[[反证过滤]]：再把“故事”如何升级成“我就是对的”看完整。
5. [[渴求共识]]、[[渴望掌控]]、[[第二份工作]]：最后看误区如何进入群体、系统和身份层。

## 核心逻辑

1. 人类大脑的很多默认设置，本来是为简单、缓慢、线性因果的世界发展出来的。
2. 现代环境更复杂、更快、更互联，也更难靠单一视角看清。
3. 旧操作系统一进入新环境，就会更依赖故事、确定感、一致、掌控和身份防御。
4. 所以问题不是“人为什么这么笨”，而是“旧脑子遇到新世界时，会系统性放大哪些误判”。
5. 真正的升级不是消灭直觉，而是让直觉重新接受复杂现实的校正。

## 建议一起看的 N 卡

- [[复杂性]]：复杂性：问题的难，不只是变量多，而是变量彼此影响、反馈滞后、因果非线性，无法被单点控制和一次看清。
- [[结构性错配]]：结构性错配：问题之所以反复出现，不只是因为某个人做错了，而是环境、规则、角色、目标或操作方式本身已经彼此不匹配。
- [[简化故事]]：简化故事：大脑为了更快理解世界，会自动补全缺失信息，把复杂现实压缩成一个看起来合理、但往往过于单一的叙事。
- [[感觉正确]]：感觉正确：一种强烈的确定感，让人把“我现在非常确信”误当成“事实已经被我看清”。
- [[渴求共识]]：渴求共识：在冲突、不确定和社会疼痛面前，人会本能地把“尽快一致”放到“把问题看完整”之前。
- [[渴望掌控]]：渴望掌控：面对复杂、不确定和失控感时，人会本能地想把更多东西抓回中心，以为更强控制就等于更好结果。
