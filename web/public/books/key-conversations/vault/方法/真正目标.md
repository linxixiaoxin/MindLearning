# 真正目标
> 创建时间：2026-04-22
> 最后更新时间：2026-04-22

这是一个 `N / Note` 概念卡。

## 核心定义

`真正目标`：在关键对话中，同时为自己、对方和关系寻找都值得保留的目标，并据此持续校正表达与行动方向。

## 可直接调用的判断

- 很多对话谈崩，不是因为没有技巧，而是因为目标悄悄从“解决问题”漂移成了“证明我对”“惩罚你”或“赶紧结束”。
- 真正目标不是一句空泛的“和平解决”，而是能指导你下一句怎么说、下一步怎么做的方向锚点。
- 当你能同时说清楚自己想保住什么、对方值得被保住什么、关系想保住什么时，傻瓜式选择通常就会开始松动。

## 这张卡解决什么

- 帮助你在高压对话中不被面子、愤怒和恐惧带偏。
- 帮助你把“我想赢”改写成“我真正想实现什么”。
- 帮助你在准备对话和中途失控时都能快速回到方向感。

## 进入哪些问题场景

- 一开口就变得很冲，事后发现自己根本没朝想要的结果前进。
- 明明想解决问题，却总在证明自己更有道理。
- 很想沟通，但担心一说就伤关系，于是开始退缩或拖延。

## 场景动作模板

### 关键对话前怎么用

- 先问自己：我希望为自己实现什么？
- 再问：我希望为对方实现什么？
- 再问：我希望为我们的关系实现什么？
- 最后问：如果这些真是我的目标，我该怎么开口？

### 对话中失控时怎么用

- 一旦发现自己开始想赢、想刺痛对方或想赶紧逃开，先暂停几秒。
- 直接承认：我的目标已经偏了。
- 重新说出真正目标，再决定继续、重说还是先修安全感。

## 红线边界

- 不要把“真正目标”说成只有对自己有利的目标。
- 不要把它用成委曲求全的遮羞布，真正目标必须包含真实问题。
- 不要只在对话前问一次，关键时刻更需要反复问。
