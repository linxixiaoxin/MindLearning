# 全书导读

> 这本书要解决的不是怎样更好地控制下属，而是如何通过教练式对话唤醒对方的觉察、责任感和行动力，让绩效提升来自内驱而不是外压。

## 这本书在回答什么

- 主问题：什么是教练式管理？
- 核心转向：当管理者习惯直接给答案，下属会逐渐失去自主思考和责任感。
- 最终落点：高绩效不只是结果高，而是人也在过程中被发展出来。

- [[教练式管理]]

- [[提问力]]

## 全书怎么推进

1. 很多绩效问题不是因为员工不听话，而是因为对话方式太控制。
2. 当管理者习惯直接给答案，下属会逐渐失去自主思考和责任感。
3. 教练式管理通过提问帮助对方看清目标、现实、选项和行动。
4. 觉察提升了，责任感和行动力才更可能被激发。
5. 长期看，教练式对话会提高个人成长速度和团队成熟度。
6. 高绩效不只是结果高，而是人也在过程中被发展出来。
