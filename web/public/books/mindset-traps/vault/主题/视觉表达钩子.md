# 视觉表达钩子

## 一、整本书可复用的视觉接口

### 1. 心智误区的由来章节图

- 适用范围：[[第一章心智误区的由来]]
- 推荐结构：心智误区的由来主结构 + 证据卡 + 底部收口
- 要表达的内容：现代复杂世界与人类旧本能之间存在结构性不匹配。
- 必须出现的对象：回环箭头、节点、流量管道、杠杆、仪表盘、透镜
- 最适合的页型：章节页、机制页、信息图页
- 为什么适合做图：它能把这一章最值钱的关系线和现实后果一起画出来。

### 2. 《走出心智误区：直面复杂世界的领导力》全书总图

- 适用范围：全书
- 推荐结构：中心总图 + 四周章节路径
- 要表达的内容：把「为什么曾经帮助人类生存的本能直觉，如今反而成了解决复杂问题的陷阱？」画成一张能继续翻页的总图。
- 必须出现的对象：人物站位、组织图、路标、旗帜、台阶
- 最适合的页型：首图、总纲页、封面页
- 为什么适合做图：适合保住整本书的主问题、总结构和章节推进。

### 3. 为了赢而倾听、为了解决问题而倾听、[[为了学习而倾听]]三层沟通升级。

- 适用范围：全书
- 推荐结构：阶梯图
- 要表达的内容：为了赢而倾听、为了解决问题而倾听、[[为了学习而倾听]]三层沟通升级。
- 必须出现的对象：人物、对话框、桥梁、箭头、气泡
- 最适合的页型：首图、总纲页、目录页
- 为什么适合做图：便于读者迅速自测所处层级。

### 4. 从[[身体信号]]、情感分析、人生目标到仁爱接纳的[[正念阶梯]]。

### 5. 冲突出现后通向[[妥协]]、两极分化或[[第三选择]]的不同路径。

- 推荐结构：十字路口图

### 6. 客观信息经过[[简化故事]]和[[感觉正确]]两层过滤后，变成个人偏见。
