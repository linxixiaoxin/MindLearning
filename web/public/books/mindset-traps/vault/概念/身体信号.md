# 身体信号

## 核心定义

`身体信号`：在我们还没把经验编成故事之前，身体已经先发出的紧张、收缩、发热、僵住、想躲开或想攻击等早期信息。

## 核心观点

- 很多误区不是从想法开始的，而是从身体已经被触发、但我们还没意识到开始的。
- 身体往往先知道“我正在进入旧模式”，大脑只是更晚才给它补上解释。
- 如果一个人长期和身体断联，就更容易把情绪、压力和威胁感直接误当成事实。
- 在《[[走出心智的误区]]》里，[[身体信号]]不是玄学补充，而是跳出自动反应的最早入口之一。

## 这个概念在解决什么

- 帮助更早识别自己什么时候已经被旧本能接管。
- 帮助把“我就是这样”改写成“我刚刚有了某种身体反应”。
- 帮助把情绪觉察从事后反省，提前到现场校准。

## 你会在什么情况下遇到它

- 一被质疑就胸口发紧、声音变硬、想立刻证明自己。
- 冲突刚开始，身体已经想退、想停、想消失，但脑子还在告诉自己“我没事”。
- 开重要会前心跳加快、肩膀僵住、呼吸变浅，结果判断质量跟着下降。
- 事后回看会发现：自己其实很早就有信号，只是当时没有把它当信息。

## 你可以怎么使用

### 现场觉察时怎么用

- 先别急着解释为什么，先描述身体到底发生了什么。
- 例如：胸口紧、胃缩、肩膀抬高、手心发热、呼吸变浅、想插嘴、想逃开。
- 这样做的价值，是把“我就是对的 / 我就是被针对”先还原成“我正在被触发”。

### 冲突里怎么用

- 当你发现身体已经明显进入防御状态时，先暂停，不要继续把那个状态包装成结论。
- 你可以先让身体回一点空间，再决定要不要继续说、怎么说。
- 这会直接降低 [[感觉正确]] 和 捍卫自我 启动的速度。

### 日常训练时怎么用

- 每天记录一两个“今天最早出现的[[身体信号]]”，比事后写一堆道理更有帮助。
- 逐渐训练自己把“[[身体信号]] -> 情绪命名 -> 反应选择”连起来。

## 使用边界

- [[身体信号]]不是事实判决，只是早期信息。
- 它告诉你“有东西发生了”，但不自动说明对方真的在攻击你、局面真的危险，或你的判断一定正确。
- 如果把[[身体信号]]直接当命令执行，也可能只是另一种被自动反应牵着走。
